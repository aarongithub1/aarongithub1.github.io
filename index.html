What is ‘void’ in a method declaration? How does this differ from declaring a method with a return type? Void' in a method declaration indicates that the method will not return a value. If you want to return a value from a method the "return type" must be indicated in the declaration.

For example: public static void testFunc(int testInt) {

Will accept one integer, received from the Function Call, and declares a local variable called "testInt." After running the block of code between the curly braces, it will return nothing.

public static double randomFunction(int x, int y) {

In this example randomFunction will accept two integers and declare int x and y as local variables.
After it has completed it's task, it will return a "double" value.
